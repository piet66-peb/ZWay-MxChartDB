Take a snapshot of a chart

A snapshot allows you to freeze the current state of a chart for later use.

This is done by cloning the chart data into a special database file called 'Snapshots':
  • Open the chart.
  • Press the 'Take Snapshot' button.
  • Choose whether you want to clone the whole chart or only a specific part.
  • Define a new, meaningful ChartId for the snapshot.
  • Press the 'OK' button.
By default, only the administrator can do this.

Snapshots can be viewed like normal charts from the chart index.
