Deleting Stored Sensor Data Completely

Deleting the stored sensor data of a chart is only possible if:
  • The owning module instance is deactivated.
  • The owning module instance is deleted.
  • The chart is a snapshot or another clone without owning instance.
To delete the chart data from the database, follow these steps:
  • Open the Chart Administration page
  • Eventually deactivate the module instance, if existing.
  • Select 'Delete chart data'.
