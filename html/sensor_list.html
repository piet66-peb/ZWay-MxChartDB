Do changes in the sensor list

You can always add more sensors to the end of the sensor list.

However once the first data is stored in the database, it is not possible to change the order of the existing sensors.

To remove a sensor you can disable it.

You can also reuse this entry by changing the device.
To ensure that this is done intentionally, it is necessary to confirm this change by checking the 'Continue current chart with new devices' box.

Or you can restart the chart with an empty status by deleting the old values.
